HTML Editor

(Week 5) I ended up choosing Sublime 3 partially because I'm not sure what functionalities I'm really looking for. I didn't run into any issues installing or using it. It is very user friendly. I found that when you close it and open it back up all of your tabs are still open. That seems like it would be really handy if you're working on a project with several pages. Also, I ran into a spot where I wanted to switch a bunch of headings from h3 to h4 and the find and replace was really helpful and easy to use.
