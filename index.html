Welcome to my Project 0 CIT 82 - Spring 2017

Thank you for visiting my Project 0 site. This site has been built from a Bootstrap template. There are links below that are provided to illustrate areas studied during the accelerated CIT 82-Web Technologies course at Fresno City College taught by Rio Waller. This course covers many topics ranging from frameworks, programming languages and web apps to the dark web and identifying the components of your own home network. I have compiled some of the topics that were particularly interesting to me and provided a bit of information to acquaint you with each topic.

 

It may be important to note that my skill level is extreme beginner. This is officially the first website I’ve built and so it is not extravagant but I have learned so much about the web, websites and the way we interact with them. The next time I run across a site or an app and a button is not working just right or a certain functionality just seems to be off I will not be critical. Creating something that seamlessly performs and looks aesthetically pleasing is no small feat and anyone who can accomplish that has my respect.

 

First, I’ll share the illustration I created in Google Drawings of my site map. This shows the intended layout for my site pages and topics. If all goes well my site will function according to this plan.
